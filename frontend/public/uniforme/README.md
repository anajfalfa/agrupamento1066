# Pasta de Imagens de Uniformes

Esta pasta contém as imagens dos uniformes de cada secção.

## Como adicionar imagens:

Adicione imagens com os seguintes nomes:

- `lobitos.png` - Imagem do uniforme dos Lobitos
- `exploradores.png` - Imagem do uniforme dos Exploradores
- `pioneiros.png` - Imagem do uniforme dos Pioneiros
- `caminheiros.png` - Imagem do uniforme dos Caminheiros

## Formato recomendado:

- Formato: PNG ou JPG
- Tamanho: Mínimo 800x800px
- Fundo: Transparente (PNG) ou branco
- Orientação: Quadrada ou vertical

As imagens serão exibidas automaticamente na página de Uniformes quando adicionadas a esta pasta.
